Spinner

A spinner can be used to display the loading state when the part of the page is waiting for the results.

Import

import Spinner from '@trendmicro/react-styled-ui/Spinner';
// or
import { Spinner } from '@trendmicro/react-styled-ui';

Usage

<Spinner />
EDITABLE EXAMPLE

Colors

<Stack direction="row" spacing="6x">
  <Spinner color="red:50" />
  <Spinner color="red" />
  <Spinner color="white" />
  <Spinner color="rgba(255, 167, 196, 1)" />
  <Spinner color="#8f41e9" />
</Stack>
EDITABLE EXAMPLE

Sizes

Use the size prop to change the size of the Spinner. You can set the value to xs, sm, md, lg, or xs. The corresponding stroke width is listed as below:

SizeStroke Width
xs2px
sm2px
md4px
lg4px
xl4px
<Stack direction="row" spacing="6x">
  <Spinner size="xs" />
  <Spinner size="sm" />
  <Spinner size="md" />
  <Spinner size="lg" />
  <Spinner size="xl" />
</Stack>
EDITABLE EXAMPLE

StrokeWidth

<Stack direction="row" spacing="6x">
  <Spinner strokeWidth={1} />
  <Spinner strokeWidth={2} />
  <Spinner strokeWidth={3} />
  <Spinner strokeWidth={5} />
  <Spinner strokeWidth={10} />
</Stack>

Props

NameTypeDefaultDescription
sizestring'md'The size of the spinner. One of: 'xl', 'lg', 'md', 'sm', 'xs'
colorstring'blue:60'The color of the spinner.
strokeWidthnumberSet a different stroke width for the spinner.
speedfloat2Set a different speed for the spinner.
