Badge

Badges are used to highlight an item's status for quick recognition.

Import

import Badge from '@trendmicro/react-styled-ui/Badge';
// or
import { Badge } from '@trendmicro/react-styled-ui';

Usage

100
<Badge>100</Badge>
EDITABLE EXAMPLE

Badge Color

  • Pass the variantColor prop to customize the color of the Badge. variantColor can be any color key with key 60 that exists in the theme.colors.

    Thats mean the green:60 exists in the theme.colors then can use variantColor="green".

  • Learn more about theming.
