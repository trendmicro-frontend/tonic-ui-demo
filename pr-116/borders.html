Borders

The border theme object is around the following scales:

  • borders
  • borderWidths
  • radii (Radius)

which is based on the Styled System Theme Specification

Border

export const borders = {
  1: '1px solid',
  2: '2px solid',
  none: 0
}
export const borderWidths = {
  1: '1px',
  2: '2px',
  none: 0
}
<Stack direction="column" spacing="1rem">
  <Stack direction="row" spacing=".5rem">
    <Box border={1} width="12x" height="12x" bg="gray:70"/>
    <Box borderTop={1} width="12x" height="12x" bg="gray:70"/>
    <Box borderRight={1} width="12x" height="12x" bg="gray:70"/>
    <Box borderBottom={1} width="12x" height="12x" bg="gray:70"/>
    <Box borderLeft={1} width="12x" height="12x" bg="gray:70"/>
  </Stack>
  <Stack direction="row" spacing=".5rem">
    <Box border={2} width="12x" height="12x" bg="gray:70"/>
    <Box borderTop={2} width="12x" height="12x" bg="gray:70"/>
    <Box borderRight={2} width="12x" height="12x" bg="gray:70"/>
    <Box borderBottom={2} width="12x" height="12x" bg="gray:70"/>
    <Box borderLeft={2} width="12x" height="12x" bg="gray:70"/>
  </Stack>
</Stack>

Border Radius

export const radii = {
  circle: '50%',
  none: 0,
  sm: '3px',
  md: '6px',
  lg: '12px'
}
<Stack direction="row" spacing=".5rem">
  <Box border={1} borderRadius="circle" width="12x" height="12x" bg="gray:70"/>
  <Box border={1} borderRadius="sm" width="12x" height="12x" bg="gray:70"/>
  <Box border={1} borderRadius="md" width="12x" height="12x" bg="gray:70"/>
  <Box border={1} borderRadius="lg" width="12x" height="12x" bg="gray:70"/>
</Stack>

Except for breakpoints, colors, and spacing, all of the keys in the theme object map to one of our core theme. All of these keys can be replaced or extended.
